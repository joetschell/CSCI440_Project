###Welcome to PacketPal!###
PacketPal is a network tool used to calculate two metrics:
    1. Packet Error Rate
    2. Packet Delay Variation

### Usage ###
```bash
sudo ./server.py -p <port> -t <testType>
```

```bash
sudo ./testClient.py -i <IP> -p <port> -c <packetCount> -t <testType>
```
